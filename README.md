より多くの環境に対応させるため、多分海外で主流と思われるunicord版をベースにしました。
導入方法はunicord版vpatchのdllと全く同じなので各自調べてください。
一応こちらの環境(1.02fのCDからインストール後v1.02hにアップデートし、dx8 to dx9とunicord版vpatchを導入)では
問題なく動作しています。

紅魔郷は、霊夢A(夢符)でP16～31のときにショット音が消滅します。
他の機体はこのような動作をしないこと、これ以外のp値ではショット音が出ることからこれはバグだと考え、修正パッチの作成に至りました。

東方紅魔郷.exeのファイルオフセット"0x7685A"が原因だと思われます。
ここは霊夢A p16～31の"bulletSoundIdx"です。ここが"FF FF"になっているため音が消えているようです。
これをおそらく正しい値だと思われる"00 00"に変えればショット音が鳴るようになるみたいです。

dllを触る環境がある人でunicord版ではない方のvpatchで同じことをしたいなら、
起動時にメモリ上のRVA 0x7685Aを同じく00 00にする処理を追加すればいいはずです。
